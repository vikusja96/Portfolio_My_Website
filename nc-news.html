Northcoders news

Full stack project

The Northcoders News project a web application I have written from scratch. For the backend part of the project, I was using PostgreSQL for the database and Node.js with ExpressJS to build a server. All APIs I wrote by TDD testing technic. The server is hosted on Heroku. The front-end project part was built by using React and CSS with fully responsive design and deployed on Netlify.

Server App

For the project, we decided to implement most of the knowledge that we get on the Northcoders Bootcamp and for each side of the project one of the new techs that we weren't learned before, such as MongoDB and React Native. For the backend, we used Node.js with Express to build a server. All APIs I wrote by TDD testing technic. For the database MongoDB with Mongoose as our data modelling library. The server is hosted on Heroku. For the frontend, we used React Native with CSS in it and Expo and Android Studio to test the application.
